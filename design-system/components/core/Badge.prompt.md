Compact inline status tag — the smaller sibling of `Pill`, sized to sit inline in table cells, list rows, and meta lines (e.g. "Stage 3 of 17").

```jsx
<Badge tone="success">Closed</Badge>
<Badge tone="primary" dot>Stage 3 of 17</Badge>
```

Same tone roles as Pill. Reach for Badge when the tag is secondary/inline; use Pill when it's a primary standalone status chip.
