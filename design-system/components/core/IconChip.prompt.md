Square rounded tinted container for an icon — anchors list rows, KPI cards, and toolbar actions. The TARS counterpart to a leading icon swatch.

```jsx
<IconChip tone="blue"><HomeIcon size={18} /></IconChip>
<IconChip tone="green" size={40} radius={12}><CheckIcon /></IconChip>
```

Tone sets both the tint background and the icon color. Pair a stroke icon (Lucide, 18px) inside. For round user marks use `Avatar` instead.
